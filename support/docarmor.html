Getting Started

Tap the “+” button on the main screen and choose “Scan Document” to use the built-in camera scanner, or “Import from Photos” to use an existing image from your library. The document is encrypted immediately after capture and stored in your local vault.
Open DocArmor. You will be prompted for Face ID or Touch ID authentication. Once authenticated, your full document vault is accessible for the current session. The vault locks automatically when you leave the app.
All documents are encrypted with AES-256 using Apple’s CryptoKit framework. The encryption key is stored in the iOS Keychain with the highest available protection class, bound to your device and your biometric or passcode. The key never leaves your device.
Yes. Tap “New Folder” from the main screen to create categories (e.g., Medical, Travel, Finance). Documents can be moved between folders at any time. All folders are encrypted with the same vault key.

Security & Privacy

Troubleshooting

If Face ID fails, DocArmor falls back to your device passcode. If neither works, ensure Face ID is set up in iOS Settings › Face ID & Passcode and that DocArmor is listed under “Other Apps.” Re-enrolling Face ID in iOS Settings can resolve persistent failures.

In-App Purchase

DocArmor is free to download. Basic features (encrypted storage, Face ID, folders) work without payment. Premium features cost $7.99 one-time to unlock — pay once and use them forever. Optional cloud backup requires a separate Sovereign subscription (purchased in Vaultyx). No recurring charges for the unlock.
Open DocArmor, tap Settings, and tap “Restore Purchases.” As long as you’re signed in with the same Apple ID used for the original purchase, it will be restored at no charge. Note: restoring the purchase does not restore your documents, which are stored locally and not backed up.
